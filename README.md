# Playwright study


## Playwright 핵심 개념
Playwright는 "웹 브라우저를 사람처럼 조작"하는 자동화 도구.

실제 사용자의 동작을 시뮬레이션

## 사용 목적
1. QA 자동화 :테스트 시나리오를 Playwright 코드로 만들고, 매일 자동으로 돌려 결과 리포트를 확인
2. 배포전 E2E 테스트 :CI/CD 파이프라인에 Playwright를 붙여

## Getting Started
###  Installation
```ts
npm init playwright@latest
npx playwright install
npx playwright test
```
